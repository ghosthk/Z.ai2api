📝 使用示例

获取模型列表：

curl -X GET "https://zai2api.ytxwz.workers.dev/v1/models" \
  -H "Content-Type: application/json"

发送聊天请求：

curl -X POST "https://zai2api.ytxwz.workers.dev/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "GLM-4.5",
    "messages": [
      {"role": "user", "content": "你好！"}
    ]
  }'

第三方软件配置：

  • API 地址：https://zai2api.ytxwz.workers.dev
  • API Key：留空或使用任意值
  • 模型：GLM-4.5

💡 提示：本代理服务完全兼容 OpenAI API 格式，可以直接替换 OpenAI 的 API 地址使用。
